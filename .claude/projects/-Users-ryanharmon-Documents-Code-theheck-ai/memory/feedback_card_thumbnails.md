---
name: No website screenshots as card thumbnails
description: Website screenshots look like iframes when used as card images — too much text, wrong visual language for a directory card.
type: feedback
---

Do not use website homepage screenshots as card thumbnail images. They contain too much text and look like embedded iframes rather than curated visual content.

**Why:** The mobile viewport screenshots of project homepages were visually cluttered — essentially a website inside a website. Cards need banner-style imagery, not page captures.

**How to apply:** When sourcing card images, look for purpose-built banner/cover images (X/Twitter profile banners, GitHub social preview images, OG images) rather than screenshots. These are designed to work as visual summaries at small sizes.
