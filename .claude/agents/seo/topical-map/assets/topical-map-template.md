## Implementation Notes

### Content Types Glossary

| Content Type | Description | Typical Length |
|--------------|-------------|----------------|
| Blog Post | Informational article | 1,500-2,500 words |
| Guide | Comprehensive resource | 3,000-5,000 words |
| Landing Page | Conversion-focused page | 500-1,500 words |
| Comparison Page | [A] vs [B] format | 2,000-3,500 words |
| Product/Service Page | Offering description | 800-1,500 words |
| Resource Page | Tools, templates, downloads | Variable |

### Priority Tier Definitions

| Tier | Definition | Action |
|------|------------|--------|
| Tier 1 | High commercial intent, core offerings, lower competition | Create first |
| Tier 2 | Supporting topics, audience problems, moderate competition | Create in months 2-3 |
| Tier 3 | Broader awareness, high competition, long-term plays | Create in months 4-6 |
